# First-Year Cash-Flow Review Template

## Purpose

Use this template when reviewing how a plan's initial observed year should map
into canonical runtime funding and benefit-payment inputs.

This template is especially important when:

- the valuation provides asset-development cash flows by class
- the ACFR provides plan-wide deductions but not class benefit payments
- the initial observed year may be broader than the later-year modeled benefit
  concept

The goal is descriptive first:

- identify what the observed year actually contains
- identify how the current or proposed build treats it
- identify whether any proxy or approximation is being used

For new plans, the preferred standard is still:

- adhere to AV treatment as much as possible
- treat any broader first-year proxy as an exception that must be justified
- prefer a simpler first cut over premature complexity; add refinement in later
  passes if needed
